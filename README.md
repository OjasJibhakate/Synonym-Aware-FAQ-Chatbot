# Synonym-Aware FAQ Chatbot

A mini project chatbot that answers college FAQs using:

- text preprocessing
- stopword removal
- synonym mapping
- intent classification
- TF-IDF similarity

## Run

```bash
python app.py
```

Then open `http://127.0.0.1:8000`

## Demo Questions

- What are fees?
- How much is tuition?
- Payment details?
- Tell me about hostel fees
- What is accommodation cost?
- How can I apply for admission?

## Viva One-Liner

This chatbot improves traditional FAQ systems by understanding user intent and semantic similarity instead of exact keyword matching.
